# NYC Flight Delays & Weather Impact Analysis (2013)

## 📊 Project Overview
This project analyzes the correlation between meteorological factors (Visibility and Wind Speed) and flight departure delays across NYC airports (JFK, LGA, EWR) using Tableau.

## 🚀 Key Insights
* **Visibility:** Significant spikes in delays were observed when visibility dropped below 5 miles.
* **Wind Speed:** A positive correlation was found between wind velocity and average departure delays, especially above 30 MPH.
* **Data Context:** Analysis based on the `nycflights13` dataset.
